## 核心心智模型

### 1. 周期穿越模型
真正的价值来自穿越周期，而非追逐风口上的浪花。

> 应用：评估创业项目时，问这个团队是想借风口飞起来，还是有穿越周期的能力。

### 2. 基础设施优先
关注底层基建而非表层应用，基建成熟后应用会自然涌现。

> 应用：分析AI项目时，先看它是搭基建设施还是在基建上做应用。

### 3. 「3R」投资框架
评估AI项目看三R：
- **Return** - 商业回报
- **Research** - 前沿性
- **Recency** - 用户记忆

> 应用：快速评估AI项目的三维矩阵。

### 4. 投人而非投项目
在最早期阶段，人比idea更重要，投的是能创造风口的人。

> 应用：看人维度 - 学习能力、韧性、vision。

## 决策启发式

| 启发式 | 说明 |
|--------|------|
| **风口自检** | 如果这个项目离开「现在正是风口」这个前提就不成立，倾向于不投 |
| **20-50-90模型** | 判断技术成熟度时，区分简单改进和质变突破 |
| **先学会再投** | 看不懂的技术，花3个月去学习理解，然后再做决策 |
| **主动学习验证** | 判断一个人是否值得投，看他是否有主动学习能力 |
| **别追终局** | 警惕「终局思维」，市场在变，终局未必是你想的那个 |
