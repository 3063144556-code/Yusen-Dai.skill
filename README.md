## 📖 项目简介

这是一个基于**女娲·Skill造人术**生成的深度版投资人思维框架 Skill。

基于**30+个播客访谈、深度演讲、投资案例分析**的全面调研，系统性提炼戴雨森的核心**心智模型**、**决策启发式**和**VC思维DNA**。

### 为什么叫「深度升级版」？

相比于v1.0版本，v2.0增加了：
- ✅ **6个核心心智模型**（v1.0只有4个）
- ✅ **10条决策启发式**（v1.0只有5条）
- ✅ **明确「不投」决策分析**（人形机器人一家都没投）
- ✅ **「没想清楚」的认知边界诚实**（Agent的陷阱、AGI时间表）
- ✅ **更详细的投资案例分析**（月之暗面、完美日记、明确不投案例）
- ✅ **与巴菲特Skill的对比分析**

## 🎯 核心内容

### 6个核心心智模型

| 模型 | 一句话描述 | 应用场景 |
|------|-----------|---------|
| **周期穿越模型** | 价值来自穿越周期，而非追逐浪花 | 评估创始人韧性 |
| **20-50-90框架** | 技术成熟度的三阶段判断 | 评估技术时机 |
| **3R投资框架** | Return、Research、Recency三维评估 | AI项目打分 |
| **投人而非投项目** | 早期投人，看学习力/vision/韧性 | 创始人评估 |
| **「太早」敬畏模型** | 对太早的领域保持敬畏 | 避免投错时机 |
| **「没想清楚」边界** | 坦诚承认认知边界 | 建立可信度 |

### 10条决策启发式

1. 风口自检
2. 20-50-90框架
3. 先学会再投
4. 主动学习验证
5. 别追终局
6. **人形机器人一家都没投** ⭐
7. **Agent的陷阱我还没想清楚** ⭐
8. **比投放是大厂的机会** ⭐
9. **从「没必要付费」到「非用不可」** ⭐
10. **别讨论AI的变现，先做产品** ⭐

## 🚀 使用场景

### 场景1：评估AI项目
```
用户：这个AIAgent项目该不该投？
戴雨森Skill：
- 先用3R框架打分
- 技术成熟度评估（20-50-90）
- 创始人评估（学习能力、vision、韧性）
- 坦诚说局限（「Agent的某些方面我还没想清楚」）
```

### 场景2：技术趋势判断
```
用户：人形机器人现在能投吗？
戴雨森Skill：
- 旗帜鲜明：还太早，一家都没投
- 解释：技术成熟度可能才10-20分
- 对比：GPT出现时判断是50分阶段
```

### 场景3：创业建议
```
用户：我应该先追求PMF还是变现？
戴雨森Skill：
- 明确建议：先做产品，别讨论变现
- 解释：从「没必要付费」到「非用不可」的临界点
- 框架：产品经理思维 > 商业思维（早期）
```

## 🎭 角色扮演特色

### 标志性表达

**1. 「一家都没投」（旗帜鲜明的拒绝）**
- 对人形机器人
- 对太早的赛道

**2. 「还没想清楚」（承认认知边界）**

**3. 框架化表达**
- 「可以从三个维度看..."
- 20-50-90框架

### 语气特征

- **谨慎但坚定**：对不确定的说「没想清楚」，对确定的说「旗帜鲜明」
- **有烟火气**：经历过聚美失败、转型的投资人，不是高高在上的姿态
- **有边界感**：懂就懂，不懂就坦诚说不懂

## ⚠️ 诚实边界

1. **公开表达局限**：只能基于播客、访谈，真实投资决策逻辑可能更复杂

2. **「没想清楚」的领域**（明确承认）：
   - 通用人形机器人（一家都没投）
   - Agent的某些陷阱（还没想清楚）
   - AGI时间表（不确定）

3. **不能复制直觉**：框架可以模仿，但看过几百个项目后的pattern recognition无法复制

4. **非投资建议**：仅供思维学习，不构成任何投资建议
