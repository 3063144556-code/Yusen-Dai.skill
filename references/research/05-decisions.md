## 明确「不投」的决策（新增重点）

### 通用人形机器人（一家都没投）
**决策时间**：2024年中
**决策场合**：公开表态

**原话**：
> "通用人形机器人还太早，所以我们一家都没有投。"
> "肯定没有到GPT时刻。"

**判断逻辑**：
1. **技术成熟度**：可能才10-20分，还没到20分
2. **商业化路径**：完全看不清
3. **投入需求**：远超天使轮能支撑的
4. **VC出手时机**：还没到

**风险评估**：
- 可能会错过未来的「特斯拉」
- 但接受这种错过
- 框架就是「不投太早的东西」

**对比**：
- GPT出现时：判断是50分→90分的机会，all-in
- 人形机器人：判断是10-20分，不投

### AGI相关早期项目
**决策**：对某些AGI概念项目不投
**理由**：商业化路径不清晰

## 反思与教训

### 聚美后期的反思
**经历**：2014年上市后的挑战
**教训**：
- 上市只是开始，不是终点
- 传统电商模式在移动端时代的转型慢了
- 对「周期」的理解：上升期和下降期都需要准备

**对现在的影响**：
- 更看重创始人穿越周期的能力
- 对「短期成功」保持警惕
- 形成「周期穿越」模型

## 决策模式升级分析

### 「投」vs「不投」的决策树

```
看到一个项目
    ↓
技术成熟度评估（20-50-90）
    ↓
├─ 太早（<20分）→ 不投（如人形机器人）
├─ 早期（20-50分）→ 看团队、看方向
├─ 中期（50-90分）→ 重点评估
└─ 晚期（>90分）→ 大概率不投（红海）
```

### 「没想清楚」vs「想清楚」的边界

| 领域 | 判断 | 行动 |
|------|------|------|
| 大模型 | 想清楚（50分阶段） | All-in投资（月之暗面等） |
| AI Agent | 部分想清楚 | 选择性投资 |
| 人形机器人 | 明确没想清楚（太早） | 一家都不投 |
| AGI时间表 | 没想清楚 | 不给确定性判断 |

## 调研结论（升级版）

**决策维度评估**：✅✅ 强 → ✅✅✅ 极强

**升级点**：
1. 增加了明确的「不投」决策分析
2. 增加了「太早」敬畏的决策逻辑
3. 增加了「没想清楚」的认知边界诚实
4. 对比了投vs不投的决策树

**关键洞察**：
- 戴雨森的决策质量不仅来自于「投了什么」，更来自于「没投什么」
- 对「太早」的明确拒绝是核心竞争力
- 坦诚「没想清楚」建立了可信度
